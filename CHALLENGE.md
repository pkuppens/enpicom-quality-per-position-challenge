# The challenge

Implement the quality per position implementation like described in the `README.md`.

It can also be implemented in a different language we have in the tech stack at ENPICOM or a hybrid of those languages.

You are expected to spend half an hour an this. If you are having a lot of fun with it you can always spend a little more, but know that our base expectation is that you have not spend more than half an hour on it. Which sets the stage to how well-crafted we expect things to be, etc.
